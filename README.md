# Exemplo de Plataforma de Simulação de Investimentos

Consiste num arquivo de excel que permite simulações simples de valores investidos, perfis de investimento e dividendos.

O usuário pode informar o salário, o rendimento da carteira, receber sugestões de valor a ser investido e porcentagens nos diferentes tipos conforme seu perfil. Também pode informar o valor que será investido e simular os dividendos de acordo com os cenários de tempo apresentados.
